## ディレクトリ内にconfig.phpを作成してください
---
```lang-php
<?php
$db['host'] = "localhost";  // DBサーバのurl
$db['user'] = "hoge"; //DBのuserid
$db['pass'] = "hoge"; //DBのpassword
$db['dbname'] = "trandferurl";  //DB名
?>
```
